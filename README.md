# Sentiment Analysis App

This is a Machine Learning-based web application that classifies product reviews as Positive or Negative.

## Features

* Text preprocessing
* TF-IDF vectorization
* Logistic Regression model
* Streamlit deployment
* Confidence score
* BERT-based advanced prediction

## Tech Stack

* Python
* Scikit-learn
* Streamlit
* NLP (NLTK, Transformers)
